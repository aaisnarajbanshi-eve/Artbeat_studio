# Artbeat Studio

Artbeat Studio is a creative sanctuary where art finds its rhythm. This repository contains the source code for the Artbeat Studio website, showcasing our artwork, murals, classes, and exhibitions.

## Repository Structure

- `artbeat-studio/`: The main React + Vite application for the website.
- `.vscode/`: Workspace settings and launch configurations.

## Getting Started

To run the project locally, please navigate to the `artbeat-studio` directory and follow the instructions in its README.

```powershell
cd artbeat-studio
npm install
npm run dev
```

For more details on the studio and its mission, visit the live site or explore the source code.
